# manyahin.ru

The source code of [manyahin.ru](https://manyahin.com) and [manyahin.com](https://manyahin.com) websites. Builded with [HUGO](https://gohugo.io/).

# commands

```bash
# run live reload server
$ npm run dev

# build public folder
$ npm run build
```
